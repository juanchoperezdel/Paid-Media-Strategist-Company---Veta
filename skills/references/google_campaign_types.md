# Campaign Types (Google Ads)

Cada tipo de campaña tiene mecánicas, best practices y limitaciones distintas. El análisis debe adaptarse al tipo.

## Tipos Principales

### Search

| Aspecto | Detalle |
| :--- | :--- |
| **Dónde aparece** | Resultados de búsqueda de Google, Search Partners |
| **Targeting** | Keywords + match types |
| **Bidding** | Todas las estrategias disponibles |
| **Best for** | Capturar demanda existente (bottom funnel) |
| **Métricas clave** | CTR, Quality Score, Search Impression Share, CPA, ROAS |
| **Estructura ideal** | SKAGs o Themed Ad Groups con keywords tight por intención |

### Performance Max (PMax)

| Aspecto | Detalle |
| :--- | :--- |
| **Dónde aparece** | Search, Display, YouTube, Discover, Gmail, Maps — todo el inventario Google |
| **Targeting** | Audience signals + asset groups (Google decide dónde mostrar) |
| **Bidding** | Solo Maximize Conversions o Maximize Conversion Value (con/sin target) |
| **Best for** | E-commerce con feed de productos, full-funnel con assets variados |
| **Métricas clave** | Conversiones, ROAS, asset performance ratings |
| **Limitaciones** | Poca visibilidad de search terms, sin Auction Insights, sin negative keywords (excepto brand exclusions a nivel cuenta) |

### Display

| Aspecto | Detalle |
| :--- | :--- |
| **Dónde aparece** | Red de Display de Google (millones de sitios, apps, YouTube) |
| **Targeting** | Audiences, topics, placements, demographics |
| **Best for** | Awareness, remarketing, alcance masivo a bajo CPM |
| **Métricas clave** | CPM, reach, frequency, view-through conversions, CTR (típicamente <1%) |
| **Nota** | CTR bajo es normal en Display — no comparar con Search benchmarks |

### Video (YouTube)

| Aspecto | Detalle |
| :--- | :--- |
| **Formatos** | Skippable in-stream, non-skippable, bumper (6s), in-feed |
| **Targeting** | Audiences, topics, channels, keywords de contexto |
| **Best for** | Awareness, consideración, remarketing con contenido visual |
| **Métricas clave** | VTR (View-Through Rate), CPV, earned actions, brand lift |
| **Nota** | Video for Action puede generar conversiones directas con CTAs |

### Shopping

| Aspecto | Detalle |
| :--- | :--- |
| **Dónde aparece** | Tab Shopping, Search results (product listings) |
| **Targeting** | Feed de productos (Merchant Center), no keywords |
| **Best for** | E-commerce con catálogo de productos |
| **Métricas clave** | ROAS, impression share, benchmark CPC, click share |
| **Nota** | La calidad del feed (títulos, imágenes, precios) impacta directamente el rendimiento |

### Demand Gen

| Aspecto | Detalle |
| :--- | :--- |
| **Dónde aparece** | YouTube, Discover, Gmail |
| **Targeting** | Lookalike audiences, custom audiences, remarketing |
| **Best for** | Mid-funnel, generación de demanda con creativos visuales |
| **Métricas clave** | CPM, CTR, conversiones, engagement rate |
| **Nota** | Reemplaza Discovery campaigns. Ideal para audiencias frías con contenido compelling |

## Implicaciones para el Análisis

- **No comparar métricas entre tipos**: CTR de Search (5-15%) vs Display (<1%) son mundos distintos
- **PMax es una caja negra**: Analizar a nivel asset group, no intentar optimizar placement por placement
- **Search es la base**: Si Search no funciona, agregar PMax/Display no lo resuelve
- **Remarketing es transversal**: Puede correr en Display, Video, o Demand Gen — evaluar incrementalidad
- **Shopping depende del feed**: Antes de optimizar bidding, asegurar que títulos, imágenes y precios estén optimizados
